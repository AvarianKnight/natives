## PLAY_PED_AMBIENT_SPEECH_NATIVE

```c
// 0x8E04FEDD28D42462 0x5C57B85D
void PLAY_PED_AMBIENT_SPEECH_NATIVE(Ped ped, char* speechName, char* speechParam);
```

Plays ambient speech; see also [`PLAY_PED_AMBIENT_SPEECH_AND_CLONE_NATIVE`](#_0xC6941B4A3A8FBBB9).

This native has a new argument on newer game builds:
* **syncOverNetwork**: indicates whether this speech should automatically be played on clones of the ped on remote machines

## Parameters
* **ped**: The ped to play the ambient speech.
* **speechName**: Name of the speech to play, eg. "GENERIC_HI".
* **speechParam**: Refer to `ConvertSpeechparamToString`
